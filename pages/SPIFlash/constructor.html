class SPIFlash
SPIFlash::SPIFlash(uint8_t cs = CS, SPIClass *spiinterface = &SPI)

Constructor Mandatory

Note

A constructor is a special kind of class member function that is executed when an object of that class is instantiated. Constructors are typically used to initialize member variables/functions of the class to appropriate default values, or to allow the user to easily initialize those member variables/functions to whatever values are desired. [*]

Parameters Optional

uint8_t cs

Refer to Defining a custom Chip Select pin

SPIClass *spiinterface

Refer to Using a non-default SPI interface

What it does

Returns Nothing

  • The constructor must be called before void setup(). The constructor can be any word of your choice. For example, the library can called by the example code below where flash can be replaced by a constructor of the user’s choice.

  • The constructor is used to call a function from the SPIFlash library.

Example code:

 1#include <SPIMemory.h>
 2
 3SPIFlash flash;   //This is the constructor. This example uses 'flash' as the constructor
 4
 5  void setup() {
 6  ...
 7  }
 8
 9  void loop() {
10  flash.readByte(...);  //The constructor 'flash' is used to call the function 'readByte()' from the library
11  ...
12  }

Related Errors None

Advanced Use

Defining a custom Chip Select pin

The library can also called by declaring the cs parameter in the constructor where cs is the uuser defined Chip Select pin for the flash module.

 1#include <SPIMemory.h>
 2
 3SPIFlash flash(33);   // The library uses the `pin 33` as the Chip Select pin instead of the default
 4
 5void setup() {
 6  ...
 7}
 8
 9void loop {
10  ...
11}

Using a non-default SPI interface [†]

  • The library currently only supports using non-default SPI interfaces on the following architectures:

    • SAMD

    • STM32

  • The csPin MUST be declared if using a non-default SPI interface.

  • Only available if library > v3.0.0

 1#include <SPIMemory.h>
 2
 3SPIFlash flash(33, &SPI1);  // The library now uses the 'SPI1' interface instead of the default 'SPI0'.
 4                            // It also uses pin 33 instead of the default Chip Select pin
 5void setup() {
 6  ...
 7}
 8
 9void loop {
10  ...
11}
