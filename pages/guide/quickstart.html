Quick Start

Goal

Get a flash chip initialized and perform one write/read cycle.

Checklist

  • Wire SPI correctly (SCK, MISO, MOSI, CS, VCC, GND)

  • Create a SPIFlash instance with your CS pin

  • Call begin() in setup()

  • Erase target region before first write

Minimal Sketch

#include <SPIMemory.h>

SPIFlash flash(10);

void setup() {
  Serial.begin(115200);
  flash.begin();

  const uint32_t addr = 0;
  flash.eraseSector(addr);
  flash.writeULong(addr, 0xDEADBEEF);

  uint32_t out = flash.readULong(addr);
  Serial.println(out, HEX);
}

void loop() {}
