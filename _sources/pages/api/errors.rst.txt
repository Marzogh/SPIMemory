Errors and Diagnostics
======================

Error Retrieval
---------------

- ``error()`` returns last error code
- ``error(VERBOSE)`` prints diagnostic detail

Common Error Classes
--------------------

- Initialization/state errors (for example ``CALLBEGIN``)
- Address and boundary errors
- Write enable and busy-state errors
- Verification failures
- Unsupported operation/chip errors

Operational Guidance
--------------------

Treat diagnostics as actionable hints. For production firmware, capture and map key error codes to telemetry or fallback behavior.
