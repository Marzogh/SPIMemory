Release Checklist
=================

Docs
----

- Update compatibility matrix
- Update API docs for new/changed methods
- Rebuild docs and verify links

Validation
----------

- Compile examples on representative boards
- Validate read/write/erase on at least one flash and one FRAM device
- Verify diagnostics still map to intended errors

Publishing
----------

- Build Sphinx HTML
- Publish output to ``gh-pages`` branch
- Confirm site availability after deploy
