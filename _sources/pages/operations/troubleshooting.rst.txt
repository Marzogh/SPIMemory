Troubleshooting
===============

No Response From Chip
---------------------

- Verify wiring and voltage levels
- Verify chip select pin assignment
- Lower SPI clock and retest
- Call ``begin()`` before any I/O

Write Failures
--------------

- Ensure erase was run before flash writes
- Disable ``HIGHSPEED`` while debugging
- Check for out-of-bounds writes and overflow mode

Intermittent Platform Issues
----------------------------

- Validate SPI bus ownership when other peripherals share the bus
- Confirm transaction mode compatibility with board core
- Test with diagnostics enabled
