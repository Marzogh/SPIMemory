# SPIMemory

SPIMemory is an Arduino library for SPI Flash and SPI FRAM chips.

This manual is organized for:

- firmware users integrating the library into applications
- maintainers extending compatibility and behavior

## Start Here

- Use [Quick Start](guide/quickstart.md) to wire, initialize, and perform first read/write.
- Use [Troubleshooting](operations/troubleshooting.md) when chips are non-responsive.
- Use [SPIFlash API](api/spiflash.md) and [SPIFram API](api/spifram.md) for the reference surface.
